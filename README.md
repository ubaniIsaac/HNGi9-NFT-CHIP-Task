# HNGi9-NFT-CHIP-Task

## CSV to JSON NFT PARSER

CSV parser converts CSV file entries to JSON format and stores them in CHIPS folder

### Installation 
Install node on pc.  
Then in CLI run 

```
$ npm install
```

### Start program by running 

``` 
$ node index.js --file (path to csv file) 
```
